# HysteriaNode

## 简介
> 聚合多个 GitHub 开源订阅源的 Hysteria2 和多协议类型
- 每 6 小时自动更新
- 多源聚合、自动去重、错误容错
- 支持 Hysteria2/Hysteria/VMess/VLESS/Trojan
